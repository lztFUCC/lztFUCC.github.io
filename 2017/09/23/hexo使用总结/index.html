我踩坑，我快乐


之前花了一天时间弄博客，在本地一切弄好了一切，就等部署到GitHub上，结果部署完后页面没有css样式，于是删了GitHub上的库，重新部署，还是没有样式；接着我不信邪，反复搞了四次，但是结果都一样。于是我在第五次新建GitHub库的时候没有勾选Initialize this repository with a README。接着部署，页面可以正常显示了。但是又有新问题了，本地的页面预览报错，部署后的页面阅读文章没有css样式，这期间我并没有做其他操作。当时感觉要炸了，一怒之下删了全部文件从头再来。
